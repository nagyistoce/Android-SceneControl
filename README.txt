The Redwood Systems Mobile Controller application enables Redwood customers to control their lights directly from an Android-enabled mobile device, running Android OS version 2.2 or later. An easy-to-use on-screen interface provides convenient lighting scene control from any location. Features include: 
• Selecting a location and adding favorite locations 
• Displaying available scenes for a selected location 
• Activation and de-activation of a selected scene
 • Password protected entry
